What We Do

We as students take care of making a web page collecting data of the official page of Statsbomb respecting all the rights of said company. We have collected all possible match filters provided by Statsbomb. We have the following filters to consult on our website:

+ Percentage of Possession: In this endpoint you must enter the id of the match, so it will return the percentage of possession of the match.

+ Matches Date: In this endpoint you must enter the id of the competition, the id of the season and the date from which you want to list all the matches.

+ Matches Date: This endpoint is responsible for listing all the points of the manager indicated in the parameter.

+ Manager with same Nationality: Endpoint is responsible for listing all managers with the same nationality.

+ Goalkeepers: This Endpoint is responsible for listing the goalkeepers who use their hands the most in the selected match.

+ Duels: This Endpoint is responsible for listing the best player of each team that wins the most duels.

+ Scorers: This Endpoint is responsible for listing all the people who score a goal in the match stopped by the match id parameter.

+ Possession: This Endpoint is responsible for listing the percentage of possession of each team in each half, including overtime.

+ Reference Player: Is responsible for showing the best valued player of the match.

+ Order Three Sixty: This Endpoint that orders the three sixty by match id.

+ Teammates Three Sixty: This Endpoint is responsible for listing all the players that appear on the camera in real time at that moment.

+ Order Competitions Ascent: This Endpoint is responsible for ordering the competitions in ascending order.

+ Female Competitions: This Endpoint is responsible for listing all women's competitions.

+ Male Competitions: This Endpoint is responsible for listing all male competitions.

2023 2DAM, Data source obtained on the official page of StatsBomb.
